# Bank Account Management System

A Java-based Bank Account Management System developed using Object-Oriented Programming concepts and JUnit Testing.

## Features

- Create Bank Account
- Deposit Money
- Withdraw Money
- Check Balance
- Account Management
- Exception Handling
- JUnit Testing

## Technologies Used

- Java
- IntelliJ IDEA
- JUnit 5

## Project Structure

src/
├── BankAccount.java
├── BankManager.java
├── BankAccountTest.java
└── Main.java
