# Bank Account Management System

## 📌 Description
The Bank Account Management System is a Java console-based application developed using Object-Oriented Programming principles and Maven. It allows users to create bank accounts, deposit and withdraw money, check account balances, and view transaction history.

## ✨ Features

- Create Bank Account
- Deposit Money
- Withdraw Money
- Check Account Balance
- View Transaction History
- Duplicate Account Validation
- Negative Amount Validation
- Custom Exception Handling
- JUnit Testing

## 🛠 Technologies Used

- Java 21
- Maven
- IntelliJ IDEA
- JUnit 5

## 🚀 How to Run

1. Clone the repository.
2. Open in IntelliJ IDEA.
3. Reload Maven dependencies.
4. Run `Main.java`.
